## Data Engineering Mini Projects

A collection of focused, hands-on data and Software engineering projects- each small enough to complete in a sitting or a weekend. Paired with a write-up to document what, why and how.

## About This Repo

This repository is a growing library of bite-sized data engineering projects. Each project explores a specific concept, tool, or pattern in the data engineering ecosystem. Every project lives alongside an article that walks through the thought process, design decisions, and lessons learned.

Whether you're brushing up on fundamentals or exploring a new tool, these projects are built to be **quick to run, easy to follow, and worth writing about.**

Projects in this repo may use any of the following - each project's README will specify exact requirements.

* **Processing**": PySpark, Pandas, dbt
* **Storage**: PostgreSQL, AWS S3, Azure Blob Storage
* **Streaming**" Kafka
* **Infrastructure**: Docker
